## Front Web para o projeto NLW

### Execução

```npm run dev```


### Configurações

Criando projeto com Vite: ```npm create vite@latest```




## Style da aplicação

Utilizando biblioteca tailwind para estruturar o style: ```npm install -D tailwindcss postcss autoprefixer```
para inicializar a configuração tailwind: ```npx tailwindcss init -p```


Componentes: 
https://headlessui.com

https://ariakit.org

biblioteca completa: https://www.radix-ui.com

utilizando Dialog: https://www.radix-ui.com/docs/primitives/components/dialog

### radix

checkbox: ```npm install @radix-ui/react-checkbox```
select: ```npm install @radix-ui/react-select```
para agrupar botoes: ```npm install @radix-ui/react-toggle-group```
